#  View 基础

## View2大职责

1. 决定自己的形态

2. 决定自己的交互

所以，本基础系列分为两部分，第一部分主要是关于View的形态的，第二部分是View的交互。

View的形态，分为“形”与“态”，一静一动。“形”主要在于View的层级、绘图等方面；“态”主要在于动画等方面。

View的交互，主要在View的动画。


### View的“形”

View的形主要有几点：

1. View的可视区域是一个矩形区域；
2. View使用“层级(Hierarchy)”的树型模型来进行界面管理；
